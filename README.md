# 📚 Simple Library Management System

Минималистичная система управления библиотечным фондом на Python, демонстрирующая принципы объектно-ориентированного программирования (ООП), такие как **наследование**, **инкапсуляция** и **полиморфизм**.

## 🛠 Функционал
Проект позволяет управлять книгами в библиотеке, включая следующие возможности:
* **Добавление книг**: Учет каждой единицы хранения с автоматическим подсчетом общего фонда.
* **Выдача и возврат**: Изменение статуса доступности книги по её уникальному ID.
* **Поиск**: Поиск доступных книг по конкретному автору.
* **Информативные отчеты**: Удобный вывод данных о книге, включая её статус («доступна» или «выдана»).

##  Архитектура классов

Программа построена на трех основных классах:

1.  **`Book`**: Базовый класс, содержащий информацию о названии, авторе, годе издания и текущем статусе.
2.  **`LibraryItem`** *(наследует Book)*: Расширяет базовый класс, добавляя уникальный идентификатор (`id`) для точного учета.
3.  **`Library`**: Класс-контроллер, который хранит список всех объектов и управляет логикой выдачи/поиска.

##  Как запустить

1. Убедитесь, что у вас установлен Python 3.x.
2. Скопируйте код в файл, например `library_system.py`.
3. Запустите скрипт через терминал:
   ```bash
   python library_system.py
   ```

##  Пример работы кода

При запуске текущего сценария вы получите следующий результат:

```text
Выдана: Мастер и Маргарита
Выдана: 1984
Возвращена: Мастер и Маргарита
Мастер и Маргарита (Булгаков, 1967) — доступна
Собачье сердце (Булгаков, 1925) — доступна
Всего книг добавлено: 3
```

## 🧪 Что было использовано из Python:
* `super().__init__` — для корректной инициализации родительского класса.
* `List Comprehension` — для эффективного поиска по списку книг.
* `Static Class Attributes` — `total_books` для ведения глобальной статистики библиотеки.
* `f-strings` — для чистого и читаемого форматирования строк.
